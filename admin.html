⚙️ 尚未连接云数据库

管理后台需要连接 Supabase 才能使用。请先：

  1. 在 supabase.com 创建免费项目；
  2. 在 SQL Editor 执行 supabase/schema.sql 与 supabase/seed.sql；
  3. 把 config.js 里的 URL 和 anon key 换成你的真实值；
  4. 在 Authentication → Users 里添加一个管理员账号(邮箱+密码)。

详见仓库 README.md。
